### 🔧 技术实现细节

#### 按钮系统逻辑
1. **访问官网**（蓝色）- 所有工具默认显示
2. **点我获取折扣**（绿色边框）- 当 `discountUrl` 字段存在时显示
3. **联系我们**（蓝色边框）- 当 `contactEmail` 字段存在时显示

#### 数据字段设计
```javascript
{
  id, name, logo, category, description,
  recommendation, useCases[], website, tags[],
  discountUrl?,  // 可选：折扣链接
  contactEmail?  // 可选：联系邮箱
}
```

#### 分类标签（data-category）
全部/AI/数据分析/项目管理/研究/市场调研/独立咨询

### 💡 经验总结
- 使用静态HTML + Tailwind CSS CDN架构简单高效，适合快速部署
- 工具数据内嵌在JS中，无需后端，维护方便
