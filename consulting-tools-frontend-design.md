# 咨询工具推荐前端 - 设计规划

## 页面定位
为咨询顾问提供专业工具推荐平台，展示AI咨询、数据分析、项目管理等工具

## 视觉风格
- **风格定位**：专业、现代、简洁、大气
- **配色方案**：
  - 主色：深蓝色 (#1a365d) - 专业、信任
  - 辅助色：渐变蓝 (#3b82f6 → #8b5cf6) - 现代科技感
  - 背景：浅灰 (#f8fafc) / 白色
  - 文字：深灰 (#1e293b) / 中灰 (#64748b)
  - 强调色：琥珀色 (#f59e0b) - 推荐标签

## 字体选择
- **标题字体**：Montserrat (Google Fonts) - 现代、有力
- **正文字体**：Inter 或 Noto Sans SC - 清晰易读
- **中文备用**：PingFang SC, Microsoft YaHei

## 页面结构

### 1. 顶部导航区
- Logo + 品牌名称
- 简单标语："为咨询顾问打造的专业工具库"
- 可能的分类筛选器（AI工具、数据分析、项目管理等）

### 2. 英雄区域
- 大标题：赋能咨询顾问的智能工具
- 副标题：精选高效工具，提升咨询效率
- 简洁的视觉元素（可能是抽象的几何图形渐变）

### 3. 工具展示区（核心）
**卡片式布局**，每个工具包含：
- 左上角：工具Logo（圆形或圆角方形）
- 标题：工具名称（大号字体）
- 标签：工具分类（如"AI分析"、"项目管理"）
- 简介：1-2句话核心功能
- 推荐理由：针对咨询场景的专属价值
- 适用场景：3-5个具体用例（图标+短描述）
- 链接按钮：访问官网 + 外部链接图标

**交互效果**：
- 卡片悬停：轻微上浮 + 阴影加深
- 链接按钮：悬停变色
- 卡片展开：点击可查看详细信息

### 4. 筛选/搜索区
- 按分类筛选：AI工具、数据分析、协作工具、研究工具等
- 搜索框：按工具名称或关键词搜索

### 5. 页脚
- 更新频率说明
- 提交工具建议的链接/表单

## 技术实现建议

### 方案A：静态站点（推荐）
- HTML + Tailwind CSS（快速构建，现代化）
- 少量JavaScript用于筛选/搜索
- 可部署在Vercel/Netlify等平台

### 方案B：React应用
- Next.js + TypeScript
- 组件化开发，便于扩展
- 支持动态数据和SEO

## 数据存储
- 工具数据存储在JSON文件中
- 结构示例：
```json
{
  "tools": [
    {
      "id": "tool-1",
      "name": "Tool Name",
      "logo": "url或本地路径",
      "category": "AI分析",
      "description": "一句话介绍",
      "recommendation": "为什么推荐给咨询顾问",
      "useCases": ["场景1", "场景2", "场景3"],
      "website": "https://...",
      "tags": ["关键词1", "关键词2"]
    }
  ]
}
```
